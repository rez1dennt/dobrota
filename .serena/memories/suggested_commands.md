Полезные команды для Windows в корне проекта:
- `start index.html` — открыть главную страницу в браузере.
- `start privacy.html` — открыть страницу политики.
- `python -m http.server 8000` — поднять простой локальный сервер, если нужен просмотр как сайта.
- `rg -n "шаблон" index.html privacy.html assets` — быстрый поиск по проекту.
- `git status --short` — проверить изменения.
В проекте не обнаружены настроенные команды сборки, тестов, линтинга или форматирования.
