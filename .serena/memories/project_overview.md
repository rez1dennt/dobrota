Проект: статический сайт стоматологической клиники «Доброта» в Рыбном. Основные страницы: `index.html` (главная) и `privacy.html` (политика конфиденциальности). Стек: HTML, CSS, vanilla JavaScript без сборщика и без backend-кода. Структура: `assets/css` для стилей (`style.css`, `privacy.css`), `assets/js` и `assets/js/modules` для клиентских скриптов, `assets/images` для изображений и логотипов. Репозиторий небольшой, изменения часто делаются точечно прямо в HTML/CSS/JS файлах.
